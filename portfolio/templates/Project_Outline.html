Technology Introduction

AWS EC2 ?

When I start this project, I would like to use a VMFusion in MacOS to make a Web service. But I couldn't waste my MacBook for only hosting.

That's why I've chosen the AWS EC2 virtual machine that help me host anytime and anywhere.

Integrated Development Environment

    OS : OS X Mojave, Linux Ubuntu 18.04 LTS
    Python 3.7.3
    DB : MariaDB, SQLite3
    UI : Bootstrap

Finishing the development in the OS X Mojave and I uploaded the files into the AWS EC2 by Github. There's one thing I sure regret. I had to develop in Docker image and insert it into the AWS EC2. It was more easiler than that I did for setting the system enviroment

Like I said in front of my page, I used Python for this project instead of Java. Because Python has some advantages than Java. Python is easy to develop and it copes with fast changes. So that I chose the Python for my first project. Of course I will start my second project with Java as soon as possible. Because it has advantages of maintenance and more systematization.

In this project, I tried to use two Databases for the comparison. SQLite is lighter than other RDBMS because it use the storage in the memory of software. So in the mass task it is improper. But it is very suitable for my first project Micro-web. In the other hand, I used also MariaDB for practicing Database Modeling and Database Normalization.

For the design of the Website, I used Bootstrap(=Framework) which makes us easiler to create, maintenance of the Website.
