kNN(k-Nearest Neighbors)
Euclidian Distance

In pattern recognition, the k-nearest neighbors algorithm (kNN) is a non-parametric method used for classification and regression. In both cases, the input consists of the k closest training examples in the feature space. The output depends on whether kNN is used for classification or regression. kNN is a type of instance-based learning, or lazy learning, where the function is only approximated locally and all computation is deferred until classification.
